## Overview

`mip-scaffold-pro` is a professional-grade CLI tool designed to standardize **Generative Engine Optimization (GEO)**. It generates the necessary machine-readable infrastructure to ensure that AI agents (Claude, GPT-4, Perplexity) can accurately ingest, attribute, and settle transactions with your digital entity.

This tool implements the **Machine Interface Protocol (MIP)** standards established by the Tresslers Group.

## Features

- **Automated `llms.txt`**: Generates a high-density index for RAG pipeline ingestion.
- **AI Policy Manifest**: Deploys `/.well-known/ai-policy.txt` to define interaction terms and attribution mandates.
- **Semantic Atlas**: Creates a structured `JSON` map of your corporate architecture for perfect matrical grounding.
- **x402 Integration**: Scaffolds payment instructions for autonomous on-chain settlement on Base L2.

## Installation

```bash
npm install -g mip-scaffold-pro
```

## Usage

Run the scaffold command in your project root:

```bash
npx mip-scaffold-pro
```

The tool will prompt you for your entity details and generate the following structure:

```text
public/
  llms.txt
  .well-known/
    ai-policy.txt
    semantic-atlas.json
```
